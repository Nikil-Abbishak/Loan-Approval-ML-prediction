# 🏦 Loan Approval & Maximum Loan Amount Prediction

## 📖 Overview
This project applies Machine Learning and Data Mining techniques to automate loan approval decisions and estimate the maximum loan amount that can be offered to approved applicants. The study is based on a real-world financial dataset containing demographic, financial, and credit-related attributes of loan applicants.

Two predictive tasks were performed:

1. **Classification** – Predict whether a loan application will be approved or rejected.
2. **Regression** – Predict the maximum loan amount that can be offered to approved applicants.

---

## 🎯 Objectives

- Prepare and preprocess a real-world loan dataset.
- Build and evaluate classification models for loan approval prediction.
- Optimize model performance through hyperparameter tuning.
- Develop an ensemble learning classifier.
- Build and compare Decision Tree regression models for maximum loan amount prediction.
- Analyze model performance and financial implications.

---
